# Memory Keeper

Memory Keeper - учебный веб-проект на React для хранения и организации личных воспоминаний пользователя.  
Проект реализован в рамках курсовой работы и использует Firebase в качестве backend-части.

## Описание проекта

На текущем этапе приложение позволяет пользователю:

- зарегистрироваться и войти в систему
- войти по `username`
- создавать, редактировать, просматривать и удалять воспоминания
- загружать несколько фотографий к одному воспоминанию
- менять порядок фотографий при создании и редактировании
- просматривать полную карточку воспоминания отдельно от режима редактирования
- возвращаться к тому же состоянию списка после просмотра или редактирования
- управлять профилем пользователя
- загружать аватар и редактировать описание профиля

Каждое воспоминание принадлежит конкретному пользователю и доступно только ему в соответствии с правилами доступа.

---

## Возможности работы с воспоминаниями

Каждое воспоминание может содержать:

- заголовок
- текст
- дату события
- дату создания
- основное место
- категорию
- тип доступа
- теги эмоций
- теги мест
- пользовательские теги
- одну или несколько фотографий

---

## Организация воспоминаний

В проекте реализованы инструменты организации архива:

- категории:
  - создание
  - переименование
  - удаление
  - защита от дубликатов без учета регистра
- сортировка:
  - по дате создания
  - по дате события
  - по алфавиту
- фильтрация:
  - по категории
  - по тегу
  - по месту
  - по диапазону дат
  - по типу доступа

---

## Поиск

В проекте реализован поиск:

- по воспоминаниям:
  - по заголовку
  - по тексту
  - по тегам
- без учета регистра
- с выводом результатов списком и кратким превью
- с подсветкой совпавших фрагментов в результатах
- по пользователям:
  - по `username`
  - с отображением аватарки или placeholder

## Интерфейс

На текущем этапе интерфейс включает:

- отдельные раскрывающиеся панели для:
  - категорий
  - поиска
  - сортировки
  - фильтров
- цветовую маркировку тегов эмоций
- отдельную страницу полного просмотра воспоминания
- карусель фотографий в полном просмотре, если у воспоминания несколько фото
- кнопки просмотра, редактирования, удаления и возврата назад

## Технологии

- **React + TypeScript**
- **Vite**
- **React Router**
- **Firebase**
  - Authentication (email/password)
  - Cloud Firestore
- CSS

---

## Авторизация и безопасность

- используется Firebase Authentication
- защищённые маршруты реализованы через `ProtectedRoute`
- доступ к данным в Firestore ограничен правилами безопасности
- пользователь имеет доступ только к своим воспоминаниям

## Структура проекта

Основные части проекта:

- `src/app` - инициализация приложения, провайдеры, роутинг
- `src/pages` - страницы приложения
- `src/features` - формы и пользовательские сценарии
- `src/widgets` - крупные интерфейсные блоки
- `src/entities` - доменная логика воспоминаний
- `src/shared` - Firebase, валидация, утилиты и общие функции

## Запуск проекта локально

1. Установить зависимости:

```bash
npm install
```

2. Создать файл `.env.local` и добавить переменные Firebase:

```bash
VITE_FIREBASE_API_KEY=...
VITE_FIREBASE_AUTH_DOMAIN=...
VITE_FIREBASE_PROJECT_ID=...
VITE_FIREBASE_STORAGE_BUCKET=...
VITE_FIREBASE_MESSAGING_SENDER_ID=...
VITE_FIREBASE_APP_ID=...
```

3. Запустить проект:

```bash
npm run dev
```
